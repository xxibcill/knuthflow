# P6-T1 - Finalize IDE Mode Decision

## Phase

[Phase 06 - IDE Expansion Optional](../../phases/phase-06-ide-expansion-optional.md)

## Objective

Decide whether Knuthflow should remain terminal-first or expand into a lightweight IDE product surface.

## Deliverables

- Product decision memo for or against editor mode
- Criteria for what Monaco would need to unlock
- Complexity and maintenance cost assessment
- Updated scope decision for the next release line

## Dependencies

- [P5-T4](../phase-05/p5-t4-create-qa-matrix-and-release-checklist.md)

## Acceptance Criteria

- The decision is explicit rather than implied
- Editor work is justified by concrete user value
- Product scope changes are documented if approved
- The team can either proceed or consciously defer
