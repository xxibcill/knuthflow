# P6-T2 - Add Monaco Editor Pane

## Phase

[Phase 06 - IDE Expansion Optional](../../phases/phase-06-ide-expansion-optional.md)

## Objective

Integrate Monaco so users can open and inspect files in-app when editor mode is justified.

## Deliverables

- Monaco editor integration in the renderer
- File open and file switch behavior
- Read-only or editable mode rules
- Layout behavior that coexists with the terminal pane

## Dependencies

- [P6-T1](./p6-t1-finalize-ide-mode-decision.md)

## Acceptance Criteria

- Files can be opened inside the app
- Editor layout does not break terminal usability
- Large files and empty states are handled acceptably
- The feature stays optional within the product surface if needed
