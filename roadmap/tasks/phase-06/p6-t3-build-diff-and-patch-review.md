# P6-T3 - Build Diff and Patch Review

## Phase

[Phase 06 - IDE Expansion Optional](../../phases/phase-06-ide-expansion-optional.md)

## Objective

Let users review changed files and patches in-app before they accept or act on changes.

## Deliverables

- Diff viewer for modified files
- Patch summary and file-level status indicators
- Review affordances for accepting, deferring, or closing review context
- Handling for multi-file changes

## Dependencies

- [P6-T2](./p6-t2-add-monaco-editor-pane.md)

## Acceptance Criteria

- Users can inspect changes without leaving the app
- The review UI works for both small and moderate multi-file changes
- File status and change context are easy to scan
- Patch review does not depend on raw git output alone

Not Implemented
