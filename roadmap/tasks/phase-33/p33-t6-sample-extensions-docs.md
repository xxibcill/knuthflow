# P33-T6 - Sample Extensions Docs

## Phase

[Phase 33 - Extension SDK and Automation Hooks](../../phases/phase-33-extension-sdk-automation-hooks.md)

## Objective

Add sample extensions and SDK documentation for supported Ralph extension workflows.

## Deliverables

- Add a minimal validator extension sample.
- Add a blueprint or scaffold sample extension.
- Add an artifact analyzer or delivery target sample if feasible.
- Document extension manifest fields, lifecycle, permissions, hooks, configuration, and diagnostics.
- Document local development and testing workflow.
- Document safety and compatibility guidelines.

## Dependencies

- Extension loader and at least one core extension point are implemented.

## Acceptance Criteria

- A developer can build and load a sample extension from docs.
- Samples demonstrate permissions and error handling.
- Documentation explains what is stable versus experimental.
- Sample extension failures are easy to diagnose.
