# P3-T1 - Build Workspace Management

## Phase

[Phase 03 - Workspaces and Sessions](../../phases/phase-03-workspaces-and-sessions.md)

## Objective

Let users register, select, and reopen local workspaces so Claude Code sessions launch in the right project context.

## Deliverables

- Workspace add and remove flow
- Recent workspace list
- Workspace selection UI before starting a session
- Validation for missing or moved directories

## Dependencies

- [P2-T4](../phase-02/p2-t4-launch-and-monitor-claude-code-runs.md)

## Acceptance Criteria

- A user can create and reopen a workspace entry
- Invalid paths are detected cleanly
- The selected workspace is available to the launch flow
- Recent workspace behavior reduces repeated setup

Not Implemented
