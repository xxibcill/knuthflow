# P20-T2 - Blueprint UI

## Phase

[Phase 20 - Skill Library and Blueprint System](../../phases/phase-20.md)

## Objective

Build blueprint browser, author, and import UI in the desktop renderer for managing blueprint library.

## Deliverables

- Blueprint browser: grid/list view of available blueprints with search and category filter
- Blueprint author: form-based blueprint editor for creating/editing blueprints with live preview
- Blueprint detail view: shows blueprint metadata, versions, usage stats
- Import/export buttons in blueprint browser
- Blueprint usage tracking: how many projects were bootstrapped from each blueprint

## Dependencies

- P20-T1 (Blueprint DB Schema) complete

## Acceptance Criteria

- Operator can browse all blueprints with search and filter
- Blueprint author form creates/edits blueprints
- Import/export buttons functional
- Blueprint usage stats visible in detail view
