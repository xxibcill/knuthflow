# P1-T1 - Scaffold Electron Shell

## Phase

[Phase 01 - Foundation and Desktop Shell](../../phases/phase-01-foundation-and-desktop-shell.md)

## Objective

Create the baseline Electron application using React, TypeScript, Tailwind, and Electron Forge so later work lands on a stable shell.
using latest version as possible

## Deliverables

- Electron main, preload, and renderer entry points
- React application bootstrapped in the renderer
- TypeScript configuration shared across app layers where appropriate
- Tailwind wired into the renderer UI
- Electron Forge configured for local development and packaging

## Dependencies

- None

## Acceptance Criteria

- The app runs locally with one command
- The main window loads the renderer successfully
- Build and dev commands are documented
- The shell can be extended without reworking the initial scaffolding
