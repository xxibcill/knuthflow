# P22-T8 - Settings About Copy

## Phase

[Phase 22 - Ralph Brand and Shell](../../phases/phase-22-ralph-brand-and-shell.md)

## Objective

Update settings, diagnostics, and about copy so support surfaces use Ralph product language while preserving technical detail.

## Deliverables

- Update About settings to show Ralph brand, version, and product description.
- Update diagnostics labels that describe Claude Code as the local execution dependency.
- Update terminal/profile settings descriptions if they imply terminal-first product scope.
- Preserve exact technical values such as executable path, Electron version, Node version, and platform.
- Audit settings tabs for old product names or wrapper-centric descriptions.
- Ensure support copy remains concise and actionable.

## Dependencies

- Visible brand metadata task is complete or in progress.
- Settings and diagnostics components remain accessible in the app shell.

## Acceptance Criteria

- About screen no longer describes the app as only a desktop wrapper.
- Diagnostics still provide enough information to debug runtime issues.
- Settings copy is consistent with Ralph-first terminology.
- No settings functionality is removed by this copy update.
